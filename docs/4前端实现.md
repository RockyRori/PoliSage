# 前端说明

VUE+TypeScript

## 初始化

```bash
node --version
```

(.venv) PS C:\Project\DocMind\frontend> node --version

v22.17.1

```bash
npm init vue@latest
```

输入项目名称为frontend，
按下空格选择Typescript之后回车

复制当前项目的package.json

```bash
cd frontend
npm install
```

## 启动前端服务

```bash
cd frontend
npm run dev
```

## frontend/dist文件夹存放前端页面

index.html布局组件
index.js定义函数
